# Journal builder

# ```Command + T ``` 3x to open 3 fresh tabs     

1. In one tab:  
   1. ```cd api```  
   2. ```json-server -p 8088 database.json``` 
  
2. In the second tab: 
   1. ```cd dist```
   2. ``` hs -o ``` 

2. Install dependencies and start the *grunt* task runner
   1.  ```cd src/lib```
   2.   ```npm install```
   3.   ```npm start```
   4.   ```grunt```

# C27 daily journal builder
![C27icon](/c27.jpeg)


## To start-up this project:
1. clone or fork this repository
1. cd api ```json-server -p 8088 database.json```
1. cd dist ``` hs -o ```
1. cd src/lib ```npm install```
1. ```npm start```
1. ```grunt```
